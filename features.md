# SGPayroll - Project Features

Based on the project structure, routes, and database schema, here is a comprehensive list of all the features available in the SGPayroll application:

## 1. Authentication & Authorization
- **User Login:** Secure authentication system for users.
- **Role Management:** Different user types to handle various administrative levels.

## 2. Employee Management
- **Employee Profiles:** Complete management of employee details, including status (Active/Inactive), tax status, and categories.
- **CRUD Operations:** Add, update, view, and delete employee accounts.
- **Bulk Import:** Upload multiple employee records simultaneously using Excel files.
- **Attendance & Leaves:** Track attendance, computation for daily logs, and manage leaves (Sick leave, vacation leave).
- **Salary Information:** Manage salary details, daily rates, and salary status.

## 3. Department & Organization Structure
- **Department Management:** Create and manage main departments.
- **Sub-groups:** Add sub-departments or groups within departments.
- **Payroll Periods:** Set and update specific payroll periods and types for each department.

## 4. Payroll Processing
- **Standard Payroll Computation:** Compute basic payroll per department.
- **Quick Payroll:** Fast payroll processing via Excel import feature.
- **13th Month Pay:** Dedicated module for calculating 13th-month pay.
- **Holiday Pay:** Computation for special holidays and minimum wage logic.
- **Data Adjustments:** Ability to edit and update previous payroll data.

## 5. Deductions & Government Contributions
- **SSS (Social Security System):** Automatic computation and tracking of SSS contributions.
- **PhilHealth:** Management of PhilHealth deductions.
- **Pag-IBIG (HDMF):** Tracking of Pag-IBIG contributions.
- **Withholding Tax:** Computation of employee withholding taxes.
- **Provident Fund:** Support for provident fund deductions.

## 6. Loan Management
- **Multi-Loan Support:** Manage various loans including SSS loans, Pag-IBIG loans, Calamity loans, and Company loans.
- **Loan Tracking:** Track loan names, original terms, interest rates, and total gross pay deductions.
- **Loan Operations:** Insert, update, delete, and view loan data for individual employees.

## 7. Payslip Generation
- **View Payslip:** Generate and view detailed payslips for employees for given payroll periods.
- **Data Requests:** Fetch detailed payslip data for review and distribution.

## 8. Reporting
- **Payroll Reports:** Generate comprehensive reports for payroll periods.
- **View Reports:** Dedicated interface for viewing and analyzing financial data across the organization.
